Inventory System

As part of my employment with American Robot Company, I was tasked with designing and creating a solution for helping the team keep track of their robot licenses.

As a result, I created an entire inventory system that allows the company to keep track of their licenses, shipped products, and product models. This project was exciting since it allowed me to learn about databases, which I had very little experience with, and to enhance my skills with JavaFX.


The database is run by MySQL and the front-end is written completely in JavaFX. I became familiar with different database queries, the use of foreign keys, Java connection statements, and result sets by reading through the detailed documentation provided by MySQL and the Oracle Java documentation.

The UI is split into three main categories. One focuses on viewing and creating new products, the second for viewing and creating models, and the last is to manage and create licenses. The application also allows license text files to be edited and downloaded so that they can be uploaded to the corresponding robot.

The database will keep track of serial and model numbers so that duplicates are avoided and to save the user time when creating a new products, models, or licenses.
