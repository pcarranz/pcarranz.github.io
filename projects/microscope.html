RFID Microscope

 Find the code for this project on GitHub.


I created this project as part of my 2-quarter senior project class. The project is also a new exhibit for the San Luis Obispo Children's Museum.

The motivation for this particular project came after observing children interact with an existing exhibit at the museum. The setup was a camera placed like a microscope that enlarges specimens embedded in acrylic blocks. Children were generally intrigued by what they saw on the screen, but did not seem truly engaged. Therefore, I sought to create a more interactive experience while providing more knowledge.


Purpose:
Create a modern, interactive exhibit that will teach children about different biological specimens through the use of technology.

Functionality:
A computer will power three RFID readers and an Arduino MEGA via serial communication. A JavaFX application will control the hardware and display information based on which RFID reader has been triggered.

The readers are placed in a row from left-to-right and labeled 1, 2, and 3. The '1' reader will display interesting facts about the selected specimen, reader '2' will show an image of the specimen under a microscope, and the third reader will play a short 20-30 second video showcasing the specimen in its natural habitat.

The Arduino is used to control 5 short LED strips that are used as indicator lights. An LED ring surrounding each reader will light up when a reader is active and there are 2 LED arrows inbetween the readers to prompt users to move on to the next reader when ready.
