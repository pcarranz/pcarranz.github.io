ZULights

 Find the code for this project on GitHub.


This project was created by a fellow computer engineer and I for a class focused on microntrollers (CPE 329). The class revolved around using a TI LaunchPad and we decided to create a night-time safety system for bicycle riders as our final design project.


Functionality:
9 LEDs and a magnetic reed switch attached to the spokes of the bike wheels are controlled by a LaunchPad. The microcontroller uses the switch to determine the speed at which the bicycle is moving and alternates which of the 9 LEDs to turn on so that there is a steady arc of light placed towards the front and back of the bike.

The project's website can be found here. I created the site as part of our marketing materials component of the project using HTML, CSS, JavaScript and Twiter Bootstrap. The site was adapted from the Agency template found at StartBootstrap.
