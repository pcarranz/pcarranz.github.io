American Robot Company's Website


I started at American Robot Company as the website developer in November of 2014. I had some experience creating small websites before I started, but this website has by far been the most extensive.

I created this website from the ground up, learning about HTML, CSS, JavaScript, and Twitter Bootstrap along the way. The current version of the website is not responsive, but I learned many design techniques about responsive design when I first started building the site. I went through all of the frustrations involved with designing responsive text and layouts, but it was nice to use the Twitter Bootstrap front-end framework which makes the process much easier.

I also learned about other related aspects of maintaining a website. For example, learning how to avoid copying the same code over and over, by using server side includes to add the navigation bar and footer to every page, and using free tools like Google Analytics and Webmaster Tools to track website traffic. Furthermore, I learned about configuring a linux server via htaccess files.

Developing this webiste also allowed me to experience the process of working with a client since I worked closely with two members of the company. I was able to receive feedback and direction while still providing my own suggestions and ideas to build a functional website that we were all happy to showcase.
